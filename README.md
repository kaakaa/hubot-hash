# hubot-hash

collect hashtags

See [`src/hash.coffee`](src/hash.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-hash --save`

Then add **hubot-hash** to your `external-scripts.json`:

```json
[
  "hubot-hash"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
